# Bible
(Teaching myself Ruby with Treehouse.com) A command-line app to look up Bible passages and display them either in terminal or browser.

## Instructions

Launchy module is required for displaying results in the browser. 
- "gem install launchy"

### To run:
- "cd path/to/repo"
- "ruby ./bible.rb"
